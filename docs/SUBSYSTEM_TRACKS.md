# Subsystem Tracks

## Track A: Memory

Versions:

```text
v195-v200
```

Must produce real page allocation and mapping behavior.

## Track B: Scheduler

Versions:

```text
v201-v206
```

Must produce real blocking and wakeup behavior.

## Track C: Storage

Versions:

```text
v207-v214
```

Must support external image or competition filesystem.

## Track D: Userland

Versions:

```text
v215-v222
```

Must support init/shell/libc-style user programs.

## Track E: Devices

Versions:

```text
v223-v230
```

Must support practical devfs and console/tty behavior.

## Track F: Network

Versions:

```text
v231-v238
```

Must support local sockets and optional real networking.

## Track G: POSIX/security

Versions:

```text
v239-v248
```

Must deepen signals, credentials, namespaces, and security.

## Track H: Architecture

Versions:

```text
v249-v258
```

Required only if the competition requires dual architecture.

## Track I: Submission

Versions:

```text
v259-v270
```

Must package evidence, reports, matrices, and final release.
