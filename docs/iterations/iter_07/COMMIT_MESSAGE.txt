controlled user fault instead of failing at load time or hanging, exposing the
next blocker: static ET_EXEC execution still needs real LoongArch virtual
address mapping for the fixed 0x120000000 load range.

and include badv in user-fault diagnostics. Do not change runtime_dispatch.rs.
