bounded non-scoring command set. The runner now executes real /musl/busybox
from sdcard-la.img for true, false, echo hello, pwd, sh -c exit, ls, and
cat /musl/busybox_cmd.txt, captures each exit code, and reports a concise
completed/attempted/matched/failed summary. It still emits no official

sdcard-backed names.
